# 📊 Dashboard de Análise Salarial em Python

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![Pandas](https://img.shields.io/badge/pandas-%23150458.svg?style=for-the-badge&logo=pandas&logoColor=white)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)

Este projeto é um dashboard interativo desenvolvido em **Python** para análise de dados salariais. A ferramenta permite visualizar tendências, médias salariais e distribuições baseadas em diferentes critérios, facilitando a compreensão do mercado de trabalho através de visualizações dinâmicas.

## 🚀 Funcionalidades
• Filtros interativos (ano, senioridade, tipo de contrato e tamanho da empresa)  
• Cálculo de métricas importantes:  
  - 💰 **Salário médio**  
  - 📈 **Salário máximo**  
  - 📊 **Total de registros**  
  - 👔 **Cargo mais comum**  
  - Gráficos interativos:  
  - Barra com top 10 cargos por salário médio  
  - Histograma de distribuição salarial  
  - Gráfico de pizza da proporção de trabalho remoto  
  - Mapa mundial de salário médio de Data Scientists por país  
  -Tabela de dados filtrados em tempo real


## 🛠️ Tecnologias Utilizadas
As seguintes ferramentas e bibliotecas foram usadas na construção do projeto:
- [Python](https://www.python.org/) - Linguagem base.
- [Pandas](https://pandas.pydata.org/) - Manipulação e tratamento dos dados.
- [Streamlit](https://streamlit.io/) - Criação da interface web interativa.
- [Plotly](https://plotly.com/python/) - Gráficos dinâmicos e de alta qualidade.

## 📦 Como executar o projeto
Para rodar este dashboard localmente, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/biafqn/Dashboard_de_salarios_python.git](https://github.com/biafqn/Dashboard_de_salarios_python.git)
2.Acesse a pasta do projeto:

    cd Dashboard_de_salarios_python

3.Instale as dependências:(Recomenda-se o uso de um ambiente virtual)

    pip install -r requirements.txt

Execute o Streamlit:

    streamlit run app.py

📂 Estrutura do Repositório
app.py: Arquivo principal contendo a lógica do dashboard e interface.
data/: Pasta contendo os arquivos CSV/Excel com os dados de salários.
requirements.txt: Lista de bibliotecas necessárias para rodar o projeto.
